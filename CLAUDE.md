# Modern Mustard Seed Website — CLAUDE.md

## What This Is
Company website for Modern Mustard Seed. Professional landing page with Three.js visual accents.

## Tech Stack
- React 19 + TypeScript
- Vite (build tool)
- Tailwind CSS 3
- Three.js (3D visuals)

## SEO & Metadata
- Full suite: meta tags, JSON-LD structured data, robots.txt, sitemap.xml
- llms.txt and .well-known/ai.txt for AI discoverability
- PWA manifest configured
- OG image is SVG — may need PNG conversion for Twitter/Facebook previews

## Build & Run
```bash
npm run dev     # Dev server
npm run build   # Production build
```
