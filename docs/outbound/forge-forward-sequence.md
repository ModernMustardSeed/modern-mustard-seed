## Guardrails (hard-coded, not aspirational)

- Verified emails only; template placeholders filtered; bounce or complaint spikes pause the cadence automatically.
- One thread per lead, replies route to the team inbox, and any reply stops the sequence instantly.
- Unsubscribe honored on first request, lead marked DNC across the whole floor.
- No invented facts: {ReasonLine} only renders from real stored evidence (audit_json, mined review, website check). Leads without evidence get a generic honest line.
- Website forge only fires after a hub open. Voice + OS pre-forge stays in the instant, token-free lanes.
